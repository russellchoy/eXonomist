#
#  Each question is a block:
#
#      @ Title | difficulty | topic | points(optional)
#      <LaTeX here — raw, exactly as in your notes, NO $ signs>
#
#    - difficulty : easy | medium | hard          (default: medium)
#    - topic      : free text                      (default: General)
#    - points     : optional integer; default by difficulty  easy=3 medium=5 hard=8
#
#  Rules:
#    - A line starting with "@" starts a new question (its header).
#    - Lines after the header, up to the next "@", are the formula. Multi-line is
#      fine for matrices/cases — use \\ for row breaks and & for column separators.
#    - Blank lines are ignored. Lines starting with "#" are comments.
#    - Write LaTeX raw (\alpha, not \\alpha). Only KaTeX-supported commands render:
#      https://katex.org/docs/supported.html
#
#  Then run  `npm run import`  — it renders every formula to catch typos and
#  regenerates lib/problems.ts. If any formula fails, nothing is written and it
#  tells you which line to fix.
#
#  Economics example (uncomment to try):
#    @ Cobb–Douglas Production Function | medium | Microeconomics
#    Y = A K^{\alpha} L^{1-\alpha}
